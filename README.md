# 🏥 CareOpt AI  
AI-Powered Healthcare Cost Optimization & Hospital Recommendation Platform

---

## 🚀 Overview

CareOpt AI is a full-stack healthcare decision-support platform that helps users find affordable hospitals based on:

- Treatment type
- City
- Budget constraints

The system provides:
- Cost comparison
- Government/Private hospital classification
- Final cost breakdown
- Scheme-based savings estimation
- Map visualization

---

## 🛠 Tech Stack

### 🔹 Backend
- FastAPI
- PostgreSQL
- AsyncPG
- Python

### 🔹 Frontend
- React (Vite)
- Tailwind CSS
- Axios
- Leaflet (Map Integration)

### 🔹 Version Control
- Git & GitHub

---

## 📊 Features

- 🔎 Treatment-based hospital search
- 📍 City-based filtering
- 💰 Budget-aware recommendations
- 🏥 Government vs Private classification
- 🧮 Final cost breakdown
- 🗺 Interactive hospital map
- 🧾 Scheme eligibility display
